# Aisyah Zainal Studio

Premium personal web design service. Build your website in 30 minutes.

## Features

- **Minimalist & Elegant Design**: Focused on conversion and user experience.
- **Fast Delivery**: Setup and launch in as little as 30 minutes.
- **Real-time Visitor Counter**: Global visitor tracking powered by Firebase.
- **WhatsApp Integration**: Seamless communication for project inquiries.
- **Responsive Layout**: Optimized for all devices using Tailwind CSS.
- **Smooth Animations**: Interactive UI elements with Motion.

## Tech Stack

- **Frontend**: React 19, Vite, TypeScript
- **Styling**: Tailwind CSS 4
- **Backend/Database**: Firebase (Auth, Firestore)
- **Icons**: Lucide React
- **Animations**: Motion

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd aisyah-zainal-studio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure Firebase:
   Create a `firebase-applet-config.json` in the root directory with your Firebase project credentials.

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/components/`: Reusable UI components.
- `src/firebase.ts`: Firebase initialization and service exports.
- `src/App.tsx`: Main application logic and layout.
- `src/index.css`: Global styles and Tailwind configuration.
- `firestore.rules`: Security rules for the Firestore database.
- `firebase-blueprint.json`: Data schema definition for the project.
